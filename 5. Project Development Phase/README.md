# Project Development Phase

## Objective
Develop an AI-powered Credit Card Approval Prediction System.

## Technologies Used
- Python
- Flask
- Scikit-learn
- IBM Watson Studio
- Pandas
- NumPy

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

## Features
- User-friendly interface
- Predicts credit card approval
- High accuracy
- Real-time prediction

## Output
The system predicts whether a credit card application will be approved or rejected.
